# Web3 Interior Edge - Design Portfolio

A modern, web3-inspired interior design portfolio website featuring cutting-edge UI/UX design and interactive elements.

## 🎨 Features

- **Modern Web3 Aesthetic**: Dark theme with vibrant gradient accents
- **Responsive Design**: Works seamlessly on all devices
- **Smooth Animations**: Scroll animations and hover effects
- **Portfolio Showcase**: 6+ featured interior design projects
- **Contact Form**: Functional contact section for client inquiries
- **Interactive Elements**: 
  - Animated gradient orbs
  - Scroll-triggered animations
  - Hover effects on portfolio items
  - Counter animations for statistics

## 🔧 Customization

### Adding Projects
Edit the portfolio grid in `index.html`:
```html
<div class="portfolio-item">
    <div class="project-image" style="background: linear-gradient(...);"></div>
    <div class="project-info">
        <h3>Your Project Title</h3>
        <p>Your project description</p>
        <span class="project-tag">Category</span>
    </div>
</div>
```

### Changing Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-dark: #0f0f1e;
    --accent-1: #667eea;
    --accent-2: #764ba2;
    /* ... etc */
}
```

### Updating Content
- Replace placeholder text with your actual content
- Update project titles and descriptions
- Modify statistics in the About section
- Add your social media links in the footer
